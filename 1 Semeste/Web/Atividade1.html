HTML

HTML é uma linguagem para estruturação e apresentação de conteúdo para a World Wide Web

HTML 5 é a mais recente evolução do padrão que define a linguagem HTML

Esta versão traz importantes mudanças quanto ao papel do HTML no mundo da Web, através de novas funcionalidades como semântica e acessibilidade

Possibilita o uso de novos recursos antes possiveis apenas com a aplicação de outras tecnologias
Muitos recursos do HTML 5 têm sido constuídos com a intenção de serem executados em dispositivos móveis como smartphones e tablets

História

Entre 1991 e 1999, o HTML se desenvolveu a partir da versão 1 até a versão 4. Em 2008, a primeira versão do HTML 5 foi lançada.

As primeiras versões do HTML foram definidas com regras sintáticas flexiveis, o que ajudou aqueles sem familiaridade com a Web. Com o tempo aumentou a tendência em tornar a sintaxe cada vez mais rígida. Apesar disso, por questões históricas (retrocompatibilidade), os navegadores ainda conseguem intepretar páginas web desenvolvidas com as primeiras versões do HTML.
